## Project Overview

This project implements and benchmarks various search and sorting algorithms for real-world systems as part of CSC 483 coursework. The assignment consists of two main questions:

### Question 1: Online Store Search Optimization
- **Sequential Search** vs **Binary Search** performance comparison
- **Hybrid Search Index** with HashMap-based name indexing
- Product catalog management with efficient insert operations
- Dataset: 100,000+ electronic products

### Question 2: Algorithm Analysis and Comparison
- Implementation of **5 sorting algorithms**: Bubble, Insertion, Merge, Quick, Heap Sort
- Comprehensive benchmarking on **5 data types**: Random, Sorted, Reverse Sorted, Nearly Sorted, Many Duplicates
- Statistical analysis of performance metrics
- Algorithm selection decision tree
